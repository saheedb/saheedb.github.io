Batch save of Event record data
AIRBAG SYSTEM (DIAGNOSTICS)
13.Batch save of Event record data
A: OPERATION
1) On «Main Menu» display, select {Each System Check}.
2) On «System Selection Menu» display, select {Airbag System}.
3) On «Airbag System» display, select {Batch save of Event record data}. After it is selected, {IMPORTANT
NOTICE} screen is displayed.
4) After checking the content of «IMPORTANT NOTICE», click the checkbox for “Yes”.
5) If the applicable law is not violated, click the [Yes] button to display the EDR data.
6) Specify the destination for the event record data to save the data there.
7) After saving the file, click the [OK] button to return the menu screen.
